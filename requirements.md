Discovery

1. Można wyświetlić listę wszystkich odkryć.
2. Można wyświetlić szczegóły pojedynczego odkrycia.
3. Można filtrować odkrycia po wybranych polach encji. Filtry są łączone operatorem AND.
4. Odkrycia może przeglądać każdy użytkownik, również niezalogowany.
5. Odkrycie może utworzyć zalogowany użytkownik.
6. Odkrycie musi mieć unikatową nazwę i url.
7. Odkrycie musi należeć do istniejącej kategorii.
8. ADMIN może edytować dowolne odkrycie.
9. USER może edytować wyłącznie własne odkrycia.
10. ADMIN może usuwać dowolne odkrycie.
11. USER może usuwać wyłącznie własne odkrycia.

Category

1. Można wyświetlić listę wszystkich kategorii.
2. Można wyświetlić szczegóły pojedynczej kategorii.
3. Można wyświetlić odkrycia przypisane do wybranej kategorii.
4. Kategorie i przypisane do nich odkrycia może przeglądać każdy użytkownik.
5. Kategorię może utworzyć wyłącznie ADMIN.
6. Nazwa kategorii musi być unikatowa.
7. ADMIN może edytować dowolną kategorię.
8. ADMIN może usunąć kategorię wyłącznie wtedy, gdy nie jest przypisana do żadnego odkrycia.

User

1. USER może wyświetlić szczegóły użytkownika.
2. Każdy może wyświetlić nazwę użytkownika.
3. ADMIN może wyświetlić wszystkich użytkowników.
4. Można wyświetlić odkrycia utworzone przez wskazanego użytkownika.
5. Można wyświetlić odkrycia edytowane przez wskazanego użytkownika.
6. Niezarejestrowany użytkownik może utworzyć konto.
7. Login i email muszą być unikatowe
8. ADMIN może utworzyć konto użytkownika.
9. ADMIN może edytować dowolnego użytkownika.
10. USER może edytować swoje dane.
11. ADMIN może usunąć użytkownika wyłącznie wtedy, gdy użytkownik nie jest autorem ani edytorem żadnego odkrycia.

Vote

1. Można wyświetlić listę wszystkich głosów.
2. Można wyświetlić szczegóły pojedynczego głosu.
3. Można wyświetlić głosy przypisane do wybranego odkrycia.
4. Głosy może przeglądać każdy użytkownik, również niezalogowany.
5. Głos może dodać wyłącznie zalogowany użytkownik.
6. Użytkownik może oddać tylko jeden głos na dane odkrycie.
7. Użytkownik nie może oddać głosu na własne odkrycie.
8. ADMIN może usunąć dowolny głos.
9. USER może usunąć wyłącznie własny głos.
10. Głos musi dotyczyć istniejącego odkrycia.
